Scan View Help

Provided that the BLE Adapter is available and turned on (the application will request to turn it on if its is off when laucnh), the BLE Device Monitor will start device discovery (scanning) automatically. Scanning will stop when any of the following conditions is met:

  • Stop button tapped
  • A device in the list tapped
  • Ten seconds elapsed since scanning started

Scanning can be restarted at any time by tapping the Scan button. During scanning RSSI values are continuously updated. When a device from the list is selected by the user, the application will attempt to connect to it. If the connection succeeds, the Device View will be launched, where details of the connected device is displayed.
