Service View Help

The Service View provides a list of the GATT characteristics of the currently selected service. Characteristics can be displayed or modified by the user. The GATT properties of the characteristic determines which operations are allowed. If the characteristic is readable, the value can be read and displayed either by tapping the characteristic in the list, or by hitting the Read button. If a characteristic supports notifications, these can be enabled/disabled by ticking the Notify check-box. Writing to a characteristic is done either by tapping directly in the value field, or hitting the Write button. In this case a HEX keyboard will be displayed. The HEX Keyboard is closed when hitting either of the Write button, Android Back, or selecting another characteristic.
